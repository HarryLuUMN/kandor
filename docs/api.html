World building

WorldBuilder.create_world(prompt=...) returns a structured blueprint.

WorldBuilder.create_memory(blueprint) seeds a TemporalKGMemory instance from the blueprint.

LLM backends

  • MockLLM for deterministic local tests and examples
  • OpenAICompatibleLLM for structured generation against OpenAI-compatible chat APIs

Memory

  • get_world_state_at(time)
  • get_entity_history(entity_id)
  • get_active_relations(entity_id, time=None)
  • get_events_between(start, end)
  • get_relevant_context(entity_ids, time, horizon)

Simulation

SimulationRunner.step(candidate_events) validates and applies one selected event.

Generation

  • LoreGenerator.summarize_world(memory=..., time=...)
  • LoreGenerator.summarize_entity(memory=..., entity_id=..., time=...)

Widgets

TemporalKGWidget(path_or_data) renders a saved KG snapshot in notebooks when the widget extras are installed.
