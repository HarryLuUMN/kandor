sim-llm-game

LLM worldbuilding with temporal KG memory

Documentation for a Python library that combines a world-building agent, temporal knowledge graph memory, a simulation engine, and lore generation.

Getting started Architecture API guide

What it does

  • Compiles user prompts into a structured world blueprint
  • Stores world state, relations, and events in temporal KG memory
  • Advances the world with a deterministic simulation loop
  • Generates readable lore summaries from structured memory

Core modules

  • agents — world setup agents such as GodAgent
  • memory — temporal KG storage and retrieval
  • simulation — step-based event application
  • generation — lore and narrative summaries

Recommended path

  1. Read Getting started for install and the demo pipeline.
  2. Read Architecture to understand the data flow.
  3. Use API guide when integrating the library into your own project.
