# Smoke outputs

This directory stores sample outputs from real smoke-test runs.

Current artifacts:
- `openai-smoke-2026-03-24.txt` — output from `examples/smoke_openai.py`
- `openai-smoke-kg-2026-03-24.json` — serialized world spec, entities, relations, events, and current time after the smoke run

These files are not meant to be canonical benchmarks. They are lightweight execution artifacts that show the pipeline really ran end-to-end.
